## 📚 今日学习

### 学到的新知识

1. **OpenClaw Memory 核心架构**
   - 双层记忆系统：`memory/YYYY-MM-DD.md` (日常) + `MEMORY.md` (长期)
   - 自动 pre-compaction memory flush 机制
   - 支持多种后端：内置 SQLite / QMD (实验性)

2. **向量搜索技术细节**
   - Hybrid Search: BM25 + Vector 混合检索
   - MMR (Maximal Marginal Relevance) 去重
   - Temporal Decay 时间衰减（半衰期30天）
   - 支持远程 embeddings: OpenAI/Gemini/Voyage/Mistral
   - 本地模式使用 node-llama-cpp

3. **QMD 实验后端**
   - BM25 + 向量 + 重排序
   - 完全本地运行，使用 Bun + node-llama-cpp
   - 自动下载 GGUF 模型

4. **记忆工具体系**
   - `memory_search` - 语义搜索，返回片段
   - `memory_get` - 读取指定文件/行范围
