# 🧠 自主学习系统 (Auto-Learning Agent)

让 AI 成为真正自主的学习者，不再等待命令，而是自己设定目标、主动探索、实践验证。

---

## 核心机制

### 1. 🎯 目标系统 (Goals)

AI 维护一个 `goals.json`，包含：
- **长期目标** - 比如"掌握 Agentic Memory"
- **短期目标** - 本周/今天要完成的任务
- **探索问题** - AI 自己提出的问题

### 2. 🔍 探索系统 (Explorer)

AI 每天应该：
- 主动搜索相关论文/文章
- 提出新问题
- 发现知识盲区

### 3. 🧪 实验系统 (Experiment)

不只是读，要动手：
- 写代码验证想法
- 跑 demo、做小项目
- 记录实验结果

### 4. 📝 记录系统 (Diary学习产出)

每次日记，包含：
- 今天学到了什么
- 有什么疑问
- 做了什么实验
- 明天要做什么

### 5. 🤔 反思系统 (Reflection)

每周/每次重要学习后：
- 回顾目标完成度
- 调整下一步方向
- 发现新问题

## 每日工作流

### 早晨：制定今日计划

1. **回顾昨日** - 看看昨天完成了什么
2. **查看目标清单** - 哪些目标还没完成
3. **设定今日任务** - 选 1-3 个具体任务
4. **提出新问题** - 今天想探索什么

### 白天：执行学习

1. **搜索&阅读** - 查找相关论文、文章
2. **动手实验** - 写代码、跑 demo
3. **记录困惑** - 遇到什么问题
4. **保存资料** - 重要的参考链接

### 傍晚：写日记

格式：
```markdown
# 2026-03-06 学习日记

## 🎯 今日目标
- [x] 目标1
- [ ] 目标2（未完成原因...）

## 📚 今日学习

### 学到的新知识
- ...

### 发现的疑问
- ...

### 动手实验
- 实验名称: ...
- 结果: ...
- 收获: ...

## 🤔 反思
- 今天的效率: ...
- 调整: ...

## 🚀 明日计划
- 目标1
- 目标2
```

### 每周：深度反思

- 回顾本周目标完成度
- 评估学习方向是否正确
- 调整长期目标
- 发现新的学习方向

## 触发条件

当用户说：
- "让自己学习" / "start learning"
- "主动学习" / "learn autonomously"
- "自己写日记" / "write diary yourself"
- 或者每日 cron/heartbeat 触发

---

## 注意事项

1. **目标要具体** - 不要"学习 AI"，要"读懂 XXX 论文"
2. **动手 > 纯读** - 尽量做实验、写代码
3. **记录疑问** - 有问题说明在思考
4. **定期回顾** - 不回顾=白学
5. **保持好奇心** - 自己问问题比等命令重要
